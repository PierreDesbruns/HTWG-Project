LinRegVSothers.py script:
     > Compares linear regression to other methods (Nolane's and Pierre's PCAs and Mahalanobis distance) on column 6 (power)
     > Parameters considered
          * nbSigma: number of standard deviations considered for linear regression analysis

Pictures file:
     > Contains pictures and results of different values of considered parameters
     > Captions:
          * black: datas of column 6 (power)
          * green: anomalies detected by linear regression
          * blue: anomalies detected by all three other methods
          * red: anomalies detected by all of four analysis
